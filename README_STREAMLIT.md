# Customer Churn Analytics - Streamlit Dashboard

Interactive web dashboard for churn prediction and analytics.

## Quick Start

```bash
# Install dependencies
pip install -r requirements_streamlit.txt

# Run the dashboard
streamlit run streamlit_app/Home.py
```

Dashboard will open in your browser at: http://localhost:8501

## Prerequisites

**FastAPI Server Must Be Running**:
```bash
# In another terminal
uvicorn app.main:app --reload
```

## Pages

### 1. Home
- Project overview
- Feature descriptions
- Navigation guide

### 2. Single Prediction
- Input customer details via form
- Get instant churn probability
- Risk categorization and recommendations

### 3. Batch Prediction
- Upload CSV file with customer data
- Bulk predictions for multiple customers
- Download results as CSV

### 4. Analytics Dashboard
- Churn probability distribution
- Top 20 high-risk customers
- Risk category breakdown
- Retention action recommendations
- Segment-wise analysis

## Configuration

### API URL
Default: `http://localhost:8000`

Change in sidebar if your API runs on different host/port.

### Data Files Required

For Analytics Dashboard, you need:
- `outputs/customer_scores.csv` - Generated by Day 4 notebook
- `outputs/retention_actions.csv` - Generated by Day 4 notebook

Run `notebooks/04_explainability_segmentation.ipynb` to generate these files.

## Troubleshooting

**Cannot connect to API**:
- Check that FastAPI is running: `uvicorn app.main:app --reload`
- Verify API URL in sidebar

**Data files not found**:
- Run Day 4 notebook to generate `customer_scores.csv` and `retention_actions.csv`

**Port already in use**:
- Change port: `streamlit run streamlit_app/Home.py --server.port 8502`
